Principais formções

Negrito

Para utilização do negrito utilizamos a tag < b >, por exemplo, essa parte esta em negrito.Assim podemos observar(não Semântico).

Nesta frase usamos um < strong >,termo em destaque, utilizado para dar sentido a alguma coisa.(Semântico)

Itálico

Nesta Frase utilizamos o < i > para deixar termo italíco , usando a tag(não semanticos)

Tambem podemos utilzar a tag < em > para colocar termos em italico, por exemplo, O mar esta belo.(Semântico)
