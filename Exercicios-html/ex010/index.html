Listas ordenadas

Utilizamos a tag < ol > para fazer listas ordenadas, elas podem seguir o type 1(numerico), A(letras maiusculas), a(letras minusculas), I(romano maiusculas), i(romanos minuscula)

  1. Manga
  2. Goiaba
  3. Gabiroba

Podemos começar a contagem de um determinado momento, utilizando

< ol type="(o tipo)" start = "(numero desejado do inicio da contagem)">

Lista não ordenada

utilizamos a tag < ul > para listas nao ordenas; elas podem ser divididas em três type:disc(padrão, bolinha toda preta), circle(uma bolinha preta com o meio oco) e square(um quadrado totalmete preenchido).

  • Goiaba
  • Banana
